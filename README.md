# Protótipo Beiramar Pescados

Este repositório contém o protótipo da aplicação Beiramar Pescados, desenvolvido com HTML, CSS e JavaScript.

## Como Abrir e Executar o Protótipo

O protótipo é uma aplicação web estática e não requer a instalação de um servidor web local ou dependências complexas.

### 1. Pré-requisitos

Você precisará apenas de um **navegador web moderno** (como Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari) para visualizar o protótipo.

### 2. Instalação

Para obter o código do protótipo, você pode clonar o repositório ou baixar o arquivo ZIP.

#### Opção A: Clonar o Repositório

Abra o seu terminal ou prompt de comando e execute o seguinte comando:

```bash
git clone [URL_DO_SEU_REPOSITORIO]
```

Substitua `[URL_DO_SEU_REPOSITORIO]` pelo endereço do seu repositório no GitHub.

#### Opção B: Baixar o Arquivo ZIP

1.  Acesse a página principal do repositório no GitHub.
2.  Clique no botão **"Code"** (Código).
3.  Selecione **"Download ZIP"** (Baixar ZIP).
4.  Descompacte o arquivo em uma pasta de sua preferência.

### 3. Execução

Após obter os arquivos, siga os passos abaixo para executar o protótipo:

1.  Navegue até a pasta raiz do projeto (onde se encontram as pastas `login` e `menu` dentro de `beiramarpescados-24.11.2025`).
2.  Localize o arquivo de entrada principal, que é o `login/login.html`.
3.  **Abra o arquivo `login/login.html` diretamente no seu navegador web.** Você pode fazer isso clicando duas vezes no arquivo ou usando a opção "Abrir com" do seu sistema operacional.

O protótipo será carregado e você poderá interagir com as telas de login e, posteriormente, com o menu principal.
